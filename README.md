# 🧠 Mood Tracker API

A simple emotion-tracking RESTful API built with Go + PostgreSQL.

## 🚀 Features

- ✅ Add new emotions
- 📖 Get all emotions
- ❌ Delete emotions
- 🛠️ Easily extendable (update, filter, etc.)
- 🌐 Ready for cloud deployment (Render / Railway)

## 🧪 Tech Stack

- Go (Golang)
- Fibre Web Framework
- GORM ORM
- PostgreSQL
- Render (for deployment)
